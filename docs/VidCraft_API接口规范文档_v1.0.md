  -------------------------------------------------------------------------------
  **参数**            **类型**   **必填**   **备注**

**0.4 通用错误码**

  -------------------------------------------------------------------------
  **code**   **说明**
  ---------- --------------------------------------------------------------
  **200**    成功

  **400**    请求参数错误

  **401**    未鉴权或 Token 已过期

  **403**    无权限（含签名验证失败）

  **404**    资源不存在

  **409**    资源冲突（如邮箱已注册）

  **429**    触发限流（每用户每分钟 ≤5 次视频生成）

  **500**    服务器内部错误

  **503**    下游 AI API 不可用
  -------------------------------------------------------------------------
